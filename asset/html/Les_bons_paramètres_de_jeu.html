Les 5 paramètres à mettre sur minecraft

Les premiers pas sur Minecraft peuvent être les plus durs, surtout sans avoir de connaissance du jeu. C'est pour cela que nous vous proposons des paramètres à mettre en jeu pour faciliter l'apprentissage des bases.

1. Changer la langue du jeu.

Avant toute chose, il faut choisir la bonne langue pour pouvoir progresser de manière optimale dans notre merveilleuse aventure.
Au lancement du jeu, nous nous dirigerons dans les paramètres, puis nous cliquerons sur Langues. Enfin nous choissirons la langue de nos cher voisins germain, l'allemand, nommée Deutsch (Deutschland)

Vous pouvez mettre aussi une option qui peut vous aider, mais n'est pas obligatoire : Le narrateur. Cette dernière vous aidera si vous n'êtes pas totalement à l'aise avec l'allemand.

2. Changer le FOV

Le FOV ou champ de vision en français est un paramètre qui permet de changer la vision du joueur.


En effet le champ de vision mis automatiquement sur le jeu peut causer quelques problèmes d'optimisations, ce qui vous empècherez de jouez convenablement et de passer un bon moment. Nous vous conseillons donc de changer le champ de vision dans les paramètres pour la descendre au minimum. Pour changer ce paramètre, il faut aller dans Options,c'est ici que vous trouverez le FOV.

3. Changer la distance d'affichage

Comme pour le champ de vision, la distance d'affichage n'est pas optimiser par le jeu automatique et peut donc causer des problèmes de performances. Il faut aller dans Options, puis Options graphiques, trouvez l'option Distance d'affichage et la mettre au minimum.

4. Changer les touches de mouvements

Les touches qui servent à se déplacer dans le merveilleux monde de Minecraft ne sont pas optimisées : elles sont trop éloignées de toutes les autres touches utilisées par le jeu. Il faut donc aller dans Options, puis dans Contrôles, et changer les touches pour mettre les déplacements sur les flèches directionnelles.

5. Inverser la souris

Quand on lance le jeu pour la première fois, et que l'on lance une partie, on remarque souvent que l'axe Y de la souris est inversé, ce qui est embêtant pour la plupart d'entre nous. Il faut donc aller dans les options pour régler ce souci, trouver l'option Inverser la souris et cliquer dessus pour mettre l'axe Y dans le bon sens.


Et voilà, avec ces paramètres, vous pourrez normalement commencer à jouer dans de bonnes conditions. Il reste évidemment des options que vous devrez changer par vous-même pour être dans le meilleur confort, mais vous avez maintenant les paramètres généraux à avoir absolument.
