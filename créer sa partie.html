Le Mode de jeux

Le mode de jeux est le premier paramètre à choisir. Il existe 3 modes de jeux :


  • Créatif
  • Survie
  • Harcore

Créatif

Le mode de jeux créatif est un mode de jeux réservé au developpeur du jeux il est interdit d'utilisation pour les joueur lambda. Si un joueur lance une partie en créatif sans être developpeur du jeux il encoure une amande de 18 000€ ainsi que 34 ans de prison cubique.

Survie

Le mode de jeux survie est un mode expérimentale pour les joueur agrée n'ayant pas peur des bug et de la difficulté, il est hautement déconseillé au nouveau joueur de lancé dans se mode de jeux.

Harcore

si minecraft est connue c'est bien grace a son mode hardcore. Ce mode est le plus simple et le plus intuitif pour les nouveau joueur il est donc conseillé de lancé une partie en hardcore pour bien débuter. Attention : ne faite pas gaffe au message lors du changement de mode de jeux, les developpeurs on inversé les texte entre survie et hardocre ce qui peut induire en erreur.
